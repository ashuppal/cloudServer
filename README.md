# cloudServer


This is a simple node.js server to EC2 instance.  

## process

1. Create a new EC2 instance, using the AWS Control Panel (GUI)
2. Connect to the EC2 instance using SSH
3. Install node.js
4. Install git
5. Clone this repository
6. Install the dependencies
7. Run the server
